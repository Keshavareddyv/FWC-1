\section*{Question}

For any set of inputs $A$ and $B$, the following circuits give the same output $Q$, except one.

Which one is it? (GATE PH 2010)

\section*{Analysis}

We simplify each circuit using Boolean algebra.

Option (b):

$Q = A \cdot \overline{B}$

Option (c):

$Q = A \cdot \overline{B}$

Option (d):

$Q = \overline{A} + B$

Hence option (d) gives a different output.
