\section*{Question}

What is the Boolean expression for the output $f$ of the combinational logic circuit of NOR gates?

\section*{Analysis}

Using NOR gate property:

$(A+B)'$

Top circuit simplifies to:

$(P+Q)(Q+R)$

Bottom circuit simplifies to:

$(P+R)(Q+R)$

Final NOR output becomes:

$f = ((Q+R)(P+Q+R))'$

Therefore,

$f = (Q+R)'$
